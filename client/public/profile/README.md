# Profile Photo

Place your profile photo here as `photo.jpg`.

Requirements:
- Filename: `photo.jpg` (exact — case sensitive)
- Recommended size: 600×800px or similar portrait ratio (3:4)
- Format: JPG or WebP (rename to photo.jpg)

The image will be displayed in a polaroid-style frame on the Hero section.
If the image is missing, a fallback initial letter will be shown automatically.
